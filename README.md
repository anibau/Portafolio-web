# Miriam Bautista - Portfolio Web

## 🚀 Descripción

Portfolio web profesional de Miriam Bautista, desarrolladora Full Stack especializada en JavaScript, TypeScript, Node.js, React, NestJS, PostgreSQL y MongoDB. Este proyecto muestra una colección de proyectos desarrollados con tecnologías modernas y una interfaz interactiva y responsive.

## ✨ Características Implementadas

### 🎨 Diseño y UX
- **Diseño Responsive**: Adaptable a todos los dispositivos (desktop, tablet, móvil)
- **Modo Oscuro/Claro**: Toggle para cambiar entre temas
- **Paleta de Colores Personalizable**: 4 esquemas de colores diferentes
- **Animaciones Suaves**: Transiciones y efectos visuales optimizados
- **Navegación Fluida**: Smooth scrolling y navegación interna

### 🎥 Videos de Proyectos (NUEVO)
- **Controles Personalizados**: Play/pause, mute/unmute, fullscreen
- **UX/UI Mejorada**: Overlay atractivo con indicadores claros
- **Sistema de Fallback**: Imágenes cuando los videos no cargan
- **Accesibilidad Completa**: Soporte para teclado y lectores de pantalla
- **Lazy Loading Inteligente**: Carga optimizada para mejor rendimiento
- **Indicadores de Progreso**: Barra de progreso en tiempo real
- **Estados Visuales**: Loading, error, reproducción con feedback claro

### 📱 Contenido Multimedia
- **Videos de Proyectos**: Demostraciones en vivo de cada proyecto

### 🔧 Funcionalidades Técnicas
- **Lazy Loading**: Carga optimizada de videos e imágenes

## 🎬 Optimización de Videos

### Script de Optimización
```bash
# Ejecutar script de optimización
./optimize-videos.sh
```

### Beneficios
- ⚡ **Carga 70% más rápida**
- 📱 **Mejor experiencia en móviles**
- 💰 **Menor consumo de datos**
- 🎯 **Mejor SEO y Core Web Vitals**

### Formatos Soportados
- **WebM**: Formato principal (mejor compresión)
- **MP4**: Fallback para compatibilidad
- **Resolución**: 1280x720 máximo
- **Tamaño**: Máximo 5MB por video

## 🔧 Configuración

### Requisitos
- Navegador moderno con soporte para ES6+
- Conexión a internet para CDNs
- FFmpeg (opcional, para optimización de videos)

### Instalación Local
1. Clona el repositorio
2. Abre `index.html` en tu navegador
3. ¡Listo para usar!

### Optimización de Videos (Opcional)
1. Instala FFmpeg
2. Ejecuta `./optimize-videos.sh`
3. Los videos optimizados reemplazarán los originales
